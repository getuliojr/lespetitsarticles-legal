# Les Petits Articles — Legal Pages

GitHub Pages site hosting the legal pages for the **Les Petits Articles** app.

## Pages

- **Home**: https://getuliojr.github.io/lespetitsarticles-legal/
- **Privacy Policy**: https://getuliojr.github.io/lespetitsarticles-legal/privacy-policy.html
- **Delete Account**: https://getuliojr.github.io/lespetitsarticles-legal/delete-account.html

## Setup

GitHub Pages is served from the `main` branch root. Go to:
**Settings → Pages → Source → Deploy from branch → `main` / `/ (root)`**
